¿Qué es Freepto?

Freepto es un sistema GNU/Linux completo adaptado para memorias USB. Esto significa que puedes llevar la memoria USB siempre contigo y utilizar cualquier ordenador como si fuera tu portátil. Además, los datos que guardes dentro de la memoria serán automáticamente cifrados y no podrán ser leídos sin su contraseña.

La idea fundamental del desarrollo de Freepto es ofrecer un sistema sencillo con el que gestionar de forma segura las herramientas comunmente utilizadas por los activistas, sin renunciar a la comodidad de un sistema operativo tradicional.

¿Cómo se usa?

Usar Freepto es muy sencillo.

Inserta la memoria USB en cualquier ordenador, reinícialo o enciéndelo y selecciona arrancar desde USB. Entonces Freepto se iniciará.

Para consultar toda la documentación sobre cómo instalar, arrancar o utilizar Freepto, visita la página de documentación.

Siempre contigo Un sistema operativo de bolsillo

Freepto es un sistema operativo completo dentro de una memoria USB. Puedes usarla en cualquier ordenador y tendrás todo lo que necesitas. No se trata de un sistema para uso solo en casos de emergencia: Freepto está adaptado para un uso habitual.

Cifrada para custodiar tus datos

Todo el contenido de Freepto está cifrado con una contraseña que solo tú conoces. De esta manera, si pierdes la memoria USB o te la roban, no tienes que preocuparte por que tus datos estéen manos ajenas.

Todo incluido desde el correo hasta el diseño gráfico

Freepto contiene muchos de los programas útiles: navegador, gestor de correos, editor de imágenes... y si te falta algo puedes instalarlo gracias a Synaptic, el gestor de paquetes usado también en Debian y Ubuntu.

Configurada para la seguridad

Freepto es fácil de usar pero no insegura.

  • Los programas de chat y filezilla están configurados para el uso de Tor y poder así disponer de conexiones anónimas y seguras.
  • Firefox incluye las extensiones para cifrar la comunicació con los servidores lo máximo posible.
  • Podrá navegar por las paginas web .onion (paginas internas a la red Tor de las cuales no se conoce su ubicación).

Seguridad adicional opcional, pero a tu alcance

Aumentar el propio nivel de seguridad es opcional pero sencillo, ya que hemos incluido dentro de Freepto una serie de herramientas que lo facilitan:

  • Borrado seguro de archivos
  • Truecrypt, para gestionar archivos cifrados
  • Torbrowser-launcher, para tener siempre la última versión de Torbrowser y navegar de manera anónima
  • GPG, para el intercambio de correos cifrados
  • TorTP, para forzar el uso de Tor en todas las aplicaciones de red
  • Florence, para disponer de un teclado virtual donde insertar las contraseñas de forma segura
  • Tomb, para la gestión avanzada de archivos cifrados

Documentación incluida

Usar Freepto es sencillo, pero es siempre &uacuote;til tener la documentación a mano para consultar cualquier duda. Por eso hemos incluido en Freepto su manual, que te ayudará a usarla de la mejor manera.

Hemos incluido también Crypt'r'die, un folleto que hemos producido dedicado a la defensa contra el control digital que está organizado en sencillas "recetas" con las que mejorar tu seguridad. ¡&Eacuote;chale un vistazo!

Personalízala

Si quieres, puedes crear una versión modificada de Freepto incluyendo nuevos programas o haciendo modificaciones de cualquier tipo. Freepto está basada en Debian Live Build, un conjunto de herramientas que permite generar distribuciones live personalizadas basadas en Debian. Puedes contribuir a partir de nuestro repositorio.
