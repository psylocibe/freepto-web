Downloading the certificate

To securely verify the identity of the services provided by Freepto, you need to download and install our CA on your computer.

wget http://ca.freepto.mx/ca.pem

Verifying the certificate

We strongly recommend to verify the authenticity of the certificate you downloaded. You can do so in a few easy steps:

  • Download our GPG key:

    • gpg --search-key 364B68D4

  • Verify the key fingerprint:

    • This step is needed to make sure that the GPG key you downloaded is actually our GPG key.

    Check that the fingerprint outputted by this command

    gpg --fingerprint 364B68D4

    is the same as the one shown below:

    Key fingerprint = C355 97C8 7BE0 8D11 A5D8 4087 909C 6C75 364B 68D4

  • Verify the certificate:

    • Download the certificate signature:

    wget http://ca.freepto.mx/ca.pem.sig

    Now verify the certificate with GPG:

    $ gpg --verify ca.pem.sig ca.pem
    gpg: Signature made Mon 03 Nov 2014 23:45:25 GMT using RSA key ID 364B68D4
    gpg: Good signature from "Freepto Repository "

Installing the certificate

To install the certificate on your computer, just follow the steps below:

$ sudo -s
# apt-get install ca-certificates
# wget -O /usr/local/share/ca-certificates/freepto.pem http://ca.freepto.mx/ca.pem
# update-ca-certificates
